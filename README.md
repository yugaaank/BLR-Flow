## ✨ Features

- 🧩 **Monorepo layout** — `frontend/` and `backend/` in one place
- 🚌 Focused on Bengaluru public-transit flows
- 🔌 API-backed so the UI stays thin and fast

## 🚀 Quick start

```bash
cd backend && npm install && npm run dev
cd ../frontend && npm install && npm run dev
```

## 📁 Structure

```
BLR-Flow/
├── frontend/   # UI
└── backend/    # API / services
```
